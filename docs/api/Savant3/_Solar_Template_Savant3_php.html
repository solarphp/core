Provides an object-oriented template system for PHP5.

Savant3 helps you separate model logic from view logic using PHP as the template language. By default, Savant3 does not compile templates. However, you may pass an optional compiler object to compile template source to include-able PHP code. It is E_STRICT compliant for PHP5.

Please see the documentation at http://phpsavant.com/, and be sure to donate! :-)
